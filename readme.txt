Do zbudowania aplikacji można użyć dla Windows:
skryptu windows_skrypt.ps1 w PowerShellu
dla Linux:
skryptu linux_skrypt.sh

Aplikacje można zbudować również poprzez komendy:
docker-compose build
docker-compose up

Korzystanie ze strony wymaga rejestracji użytkownika.

Druga aplikacja znajduje się w folderze DrugaAplikacja - PAMiW_SecondApp.exe
Można zalogować się do niej za pomocą tych danych:
Login - admin
Hasło - admin
By aplikacja działała, wpierw powinna zostać wykonana procedura budowania aplikacji poprzez docker-compose.
